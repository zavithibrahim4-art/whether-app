# Weather Application

A responsive weather application that retrieves real-time weather information using the OpenWeatherMap API. The application provides current weather details for any searched city and remembers the last searched location using Local Storage.

## Features

* Search weather by city
* Current temperature
* Weather condition
* Humidity
* Wind speed
* Weather icon
* Loading indicator
* Error handling for invalid city names
* Saves the last searched city using Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API
* Local Storage

## Installation

Clone the repository:

```bash
git clone <https://github.com/zavithibrahim4-art/whether-app>
```

Open the project folder and launch `index.html` using Live Server or any local web server.

## API

This project uses the **OpenWeatherMap API** to retrieve live weather information.

## Assumptions

* Internet access is required.
* A valid OpenWeatherMap API key is needed.

## Known Limitations

* Weather data depends on API availability.
* Free API plans may have request limits.
* Forecast availability depends on the selected API endpoint.
